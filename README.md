# Sistema de Automação Digital - Gestão de Peças

## Funcionamento
Este sistema em Python auxilia no controle de produção e qualidade de peças industriais. Ele permite cadastrar peças, avaliar automaticamente se estão aprovadas (peso 95-105g, cor azul/verde, comprimento 10-20cm), armazenar as aprovadas em caixas de 10 peças e gerar relatórios.

## Como rodar
1. Tenha Python 3 instalado.
2. Baixe o arquivo `sistema_pecas.py`.
3. Abra o terminal na pasta do arquivo.
4. Execute: `python sistema_pecas.py`
